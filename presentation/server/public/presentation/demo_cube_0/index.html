1er cube fixe et sans éclairage : 300 lignes de code

3 étapes principales :

  1. Définir les buffers & les matrices (CPU)
  2. Calculer la position des vertices (Vertex shader, GPU)
  3. Calculer la couleur des fragments (Fragment shader, GPU)

En code, ça donne :

  1. Initier le contexte WebGL
  2. Définir les buffers des meshes (objets) à afficher
  3. Charger les shaders en mémoire
  4. Pour chaque boucle d'affichage et chaque mesh :
    • calculer les matrices (world, view, ...)
    • charger les shaders du mesh
    • passer les paramètres
    • charger les buffers
    • lancer l'affichage
