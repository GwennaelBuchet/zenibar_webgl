Etape 2 : Définir les buffers & les matrices

2. Définir les faces (i.e. l'ordre des vertices à afficher, par groupe de 3) :

    // Buffer for the cube's vertices indices.
    const indicesBuffer = gl.createBuffer();

    // indices of vertices for each face
    const indices = [
        0, 1, 2,   1, 2, 3  // front face
        ...
    ];

    // Bind to the indicesBuffer
    gl.bindBuffer(gl.ELEMENT_ARRAY_BUFFER, indicesBuffer);
    // Fill the buffer with vertices positions
    gl.bufferData(gl.ELEMENT_ARRAY_BUFFER, new Uint16Array(indices), gl.STATIC_DRAW);

3. Afficher les faces (appuyer sur 'T') :

    // Let's render
    gl.drawElements(gl.TRIANGLES,
                    36 /*count (number of vertices)*/
                    gl.UNSIGNED_SHORT, // type
                    0); // offset
